# Репозиторій університетського коду

Цей репозиторій містить матеріали, лабораторні роботи та експерименти для різних предметів **Відкритого міжнародного університету розвитку людини "Україна"**.  

Тут зберігається код для навчальних завдань, практичних робіт та інших проектів, виконаних під час навчання.
