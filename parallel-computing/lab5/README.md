# Лабораторна робота №5 – Колективні комунікації MPI

У цій лабораторній роботі розглядаються **колективні операції обміну даними між усіма процесорами**:

* **AllGather** – кожен процес надсилає свої дані всім іншим і отримує повний масив від усіх.
* **AllGatherv** – варіант AllGather з можливістю різної кількості елементів від кожного процесора.
* **AllToAll** – кожен процес надсилає частину свого масиву всім іншим, і всі отримують частини від усіх.
* **AllToAllv** – варіант AllToAll з різним розміром блоків для кожного процесора.

Ціль – продемонструвати, як MPI дозволяє ефективно обмінюватися даними між усіма вузлами одночасно, без потреби у ручному керуванні send/recv.

Запуск:

```bash
gradle lab5:runMpiTestAllGather lab5:runMpiTestAllGatherv lab5:runMpiTestAllToAll lab5:runMpiTestAllToAllv
```
