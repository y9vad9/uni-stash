# Лабораторна робота №4

Цей модуль демонструє основні операції колективної та точкової комунікації в MPI на Java з використанням бібліотеки
OpenMPI.

https://github.com/user-attachments/assets/0490012c-681a-41ee-b1dc-016c13f16270

## Запуск

Для запуску будь-якого класу використовуйте Gradle:

```bash
gradle lab4:runMpi[ClassName]
```

## Класи модуля

### Lab4

* Основний приклад збору статусів та розсилки команд.
* Кожен процесор генерує статус своєї одиниці.
* Статуси відправляються процесору з рангом 0 за допомогою `gather`.
* Відповідні команди розсилаються назад усім процесорам через `scatter`.
* Використовуються масиви `status` та `command` для відображення текстових статусів і команд.
* Демонструє збір та розсилку даних між процесами MPI.

### TestGather

* Демонстрація колективної операції `gather`.
* Кожен процесор надсилає свій масив чисел процесору з рангом `np - 1`.
* Процесор-одержувач збирає всі масиви в один і виводить результат.

### TestGatherv

* Використання `gatherv` для збору масивів різної довжини.
* Процесор-одержувач може отримувати різну кількість елементів від різних процесорів.
* Демонструє гнучкіший варіант `gather`.

### TestScatter

* Демонстрація операції `scatter`.
* Процесор-джерело розсилає частини масиву іншим процесорам.
* Кожен процесор отримує частину масиву.

### TestScatterv

* Використання `scatterv` для розсилки частин масиву різної довжини.
* Кожен процесор отримує свою частину відповідно до масивів `counts` та `displacements`.
