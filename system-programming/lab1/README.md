# Лабораторна робота №1  
Консольна програма для привітання користувача та форматування дати народження

https://github.com/user-attachments/assets/2cb88c18-1b30-422b-a629-47bc7ba94df7



## Завдання

Створити консольну програму, яка запитує дані користувача та відображає введену дату народження у форматах різних країн.

## Задача

Програма повинна:
- запитати ім’я користувача та зберегти його у змінну типу `string`;
- запитати дату народження у форматі:  
  **дд мм рррр**  
  Усі значення вводяться в одному рядку через пробіл. Кожне число повинно бути збережене в окрему змінну типу `int`;
- після введення всіх даних — вивести привітання користувачу, наприклад:  
  `Nice to meet you, <name>!`
- відобразити дату народження у форматах різних країн (3–5 варіантів на вибір).

Додаткові перевірки коректності вводу не потрібні — вважаємо, що дані завжди валідні.

## Приклад форматування

```
Do you know that your birth date will have different format in different countries, e.g.:
-- Poland:   2006-05-04
-- Finland:  4.5.2006
-- Spain:    04/05/2006

```

Приклади форматів можна переглянути тут:  
https://uk.wikipedia.org/wiki/Дата

## Запуск лабораторної

Після компіляції (через `make lab1` або `make`) виконати:

```bash
./build/lab1
```

Програма запустить консольний діалог і проведе через увесь процес.
