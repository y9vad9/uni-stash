# Лабораторна робота №8

Ця лабораторна робота демонструє базову роботу з класами в C++, включно з атрибутами, методами доступу (getter/setter) та додатковими методами класу.

Програма складається з трьох завдань, що покривають роботу з об’єктами та методами класів.

## Завдання 1 – Створення об’єкта Cat

Програма:

* Створює функцію `test_cat`, яка повертає об’єкт класу `Cat`.
* Встановлює атрибут `name` рівним `"Fluffy"`.
* Встановлює атрибут `age` рівним `5`.

## Завдання 2 – Методи доступу для Dog

Програма:

* Додає до класу `Dog` методи `get_name`, `set_name`, `get_age` та `set_age`.
* У функції `main` створює об’єкт `spot`.
* Встановлює ім’я собаки `"Spot"` і вік `7`.
* Виводить у консоль значення імені та віку за допомогою getter-методів.

## Завдання 3 – Додаткові методи для Dog

Програма:

* Додає метод `get_human_years()`, який повертає вік собаки, помножений на 7.
* Додає метод `speak()`, який повертає рядок `"Woof"`.
* Використовує ці методи для виводу додаткової інформації про об’єкт `spot`.
