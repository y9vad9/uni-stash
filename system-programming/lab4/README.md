# Лабораторна робота №4

Ця лабораторна робота демонструє базову роботу з умовними операторами в C++: `if`, вкладеними `if` та оператором `switch`.


https://github.com/user-attachments/assets/ecab17fb-4456-4fd4-9429-1a984aa61f53

Ця директорія містить три окремі завдання, кожне з яких зосереджене на іншому типі логічної розгалужувальної конструкції.

Так само, як і в попередній практичній, для лінукс-середовища використовується Docker, що забезпечує однакову та передбачувану платформу для компіляції та запуску програм.

## Завдання 1 – Перевірка можливості керувати автомобілем (простий `if`)

Програма:

* Запитує у користувача його вік (`age`).
* Якщо вік **16 або більше**, виводить:
  `Yes - you can drive!`
* Якщо менше — **нічого не виводить**, як вимагає умова.

## Завдання 2 – Можливість керувати автомобілем з урахуванням наявності авто (вкладені `if`)

Програма:

* Запитує вік користувача та булеве значення `has_car` (чи є авто).
* Якщо особі **менше 16** — виводить:
  `Sorry, come back in 'n' years and be sure you own a car when you come back.`
  де `n` — скільки залишилось до 16 років.
* Якщо **16 або більше**, але **немає автомобіля** — виводить:
  `Sorry, you need to buy a car before you can drive!`
* Якщо **16 або більше** і **авто є** — виводить:
  `Yes - you can drive!`

## Завдання 3 – Визначення дня тижня (`switch`)

Програма:

* Приймає від користувача число `day_code`.
* Використовує `switch` для відображення дня тижня за таблицею:

```
0 → Sunday
1 → Monday
2 → Tuesday
3 → Wednesday
4 → Thursday
5 → Friday
6 → Saturday
інше → Error - illegal day code
```

## Файли

```bash
make lab4
./build/lab4
```

## Docker-середовище

У цій лабораторній, як і в попередніх лабораторних роботах, використовується Docker-контейнер для запуску Linux-оточення зі встановленим компілятором GCC/Clang та необхідними інструментами.
