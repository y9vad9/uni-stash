# Лабораторна робота №5

Ця лабораторна робота демонструє використання циклів `for` та `range-based for` для обчислень і роботи з векторами в C++.

Програма складається з двох завдань:

## Завдання 1 – Сума непарних чисел

Програма повинна:

* Використати цикл `for` для обчислення суми непарних цілих чисел від 1 до 15 включно.
* Зберегти кінцевий результат у змінній `sum`.
* Вивести результат на екран.

Приклад виводу:

```
Сума непарних чисел від 1 до 15: 64
```

## Завдання 2 – Кількість елементів, що діляться на 3 або 5

Програма повинна:

* Використати `range-based for` цикл для проходження по вектору цілих чисел:

```
vector<int> vec {1,3,5,15,16,17,18,19,20,21,25,26,27,30,50,55,56,58,100,200,300,400,500,600,700};
```

* Визначити, скільки елементів у векторі діляться на 3 або на 5 без залишку.
* Зберегти кінцевий результат у змінній `count`.
* Вивести результат на екран.

Приклад виводу:

```
Кількість елементів, що діляться на 3 або 5: 21
```
