# Лабораторна робота №7

Ця лабораторна робота демонструє роботу з вказівниками та масивами в C++.

Програма складається з трьох завдань:

## Завдання 1 – Обмін значень за вказівниками

Функція:

* `swap_pointers(int* ptr1, int* ptr2)` – отримує два цілочисельні вказівники і міняє місцями значення, на які вони вказують.
* Значення змінюються без використання додаткової тимчасової змінної.
* Вхід: два цілочисельні вказівники.
* Вихід: змінені значення у пам’яті, на які вказують ptr1 та ptr2.

## Завдання 2 – Множення через вказівник

Функція:

* `multiply_with_pointer(int* ptr, int multiplier)` – отримує цілочисельний вказівник і множник.
* Значення, на яке вказує ptr, множиться на multiplier.
* Вхід: цілочисельний вказівник та ціле число-множник.
* Вихід: оновлене значення за адресою ptr.

## Завдання 3 – Пошук максимального елемента масиву через вказівники

Функція:

* `findMaxElement(int* arr, int size)` – отримує масив і його розмір.
* Повертає максимальний елемент масиву.
* Вхід: покажчик на перший елемент масиву та його розмір.
* Вихід: максимальне значення в масиві.
