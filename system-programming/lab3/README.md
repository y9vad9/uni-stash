# Лабораторна робота №3

Ця робота охоплює базові операції присвоювання, арифметичні дії над змінними, логічні вирази для прийняття рішень та роботу з грошовими одиницями через цілочислові розрахунки.

https://github.com/user-attachments/assets/0cc11cf7-8cf2-48d2-b00e-ae3795eb9ddf

Програма складається з чотирьох завдань:

## Завдання 1 – Операції присвоювання

Оголошуються змінні `num1` та `num2`.
`num1` змінюється оператором присвоювання, після чого її значення копіюється до `num2`.

Мета завдання — зрозуміти базові операції роботи зі змінними.

## Завдання 2 – Арифметичні операції зі змінними

Працюємо з двома змінними: `original_number` та `number`.
Послідовно виконуються множення, додавання, віднімання, ділення та операція модуля з обов’язковим виведенням результатів після кожної дії.

Приклад частини виводу:

```
After *2: 8
After +9: 17
After -3: 14
After /2: 7
After -original_number: 3
After %3: 0
```

## Завдання 3 – Перевірка відповідності кандидата вимогам

Програма моделює процес оцінювання кандидата на вакансію водія доставки.

Перевіряються критерії:

* вік (18+ або 15+ зі згодою батьків),
* наявність дійсного номера соціального страхування,
* відсутність автомобільних аварій.

Результат:

```
You are hired!
```

або

```
Sorry, we cannot hire you!
```

## Завдання 4 – Розмін заданої суми центів

Програма запитує кількість центів та розкладає її на мінімальну кількість монет.

Приклад:

```
Enter sum in cents: 92
You can get this amount with:
dollars: 0
quarters: 3
dimes: 1
nickels: 1
pennies: 2
```

Використовується поетапне ділення та оператор `%`, щоб визначати залишок.
