# Лабораторна робота №6

Ця лабораторна демонструє використання математичних функцій округлення, користувацьких функцій, масивів, циклів та рекурсії в C++.
Усі завдання виконані в одному файлі в рамках одного проєкту. Для роботи в Linux-оточенні використовується Docker-контейнер.

## Завдання 1 – Поділ ресторанного рахунку

Програма:

* Запитує загальну суму рахунку (`bill_total`) та кількість гостей (`number_of_guests`).
* Обчислює `individual_bill = bill_total / number_of_guests`.
* Виводить рахунок для трьох локацій:

  * Округлення вниз (`floor`)
  * Округлення до найближчого цілого (`round`)
  * Округлення вгору (`ceil`)

## Завдання 2 – Конвертація температур

Програма містить дві функції:

* `fahrenheit_to_celsius(double)`
* `fahrenheit_to_kelvin(double)`

Користувач вводить температуру у Фаренгейтах, а програма виводить результат у Цельсіях та Кельвінах.

## Завдання 3 – Робота зі списком гостей

Використовуються дві функції:

* `print_guest_list(const string[], size_t)` — друкує елементи масиву.
* `clear_guest_list(string[], size_t)` — очищує список, замінюючи кожен елемент на `" "`.

У функції `event_guest_list` виконується:

1. Друк списку гостей
2. Очищення списку
3. Повторний друк

## Завдання 4 – Рекурсивна сума цифр

Реалізована функція:

* `int sum_of_digits(int n)`, яка повертає суму цифр числа.

## Файли

* `lab6_yaroshchuk_pi_22_1.cpp` — реалізація всіх завдань.
* `Makefile` — підтримка збірки через `make lab6` або `make`.

## Компіляція та запуск

```bash
make lab6
./build/lab6
```
