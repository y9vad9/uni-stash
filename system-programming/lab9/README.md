# Лабораторна робота №9

Ця лабораторна робота демонструє використання конструкторів класів у C++, включно зі стандартним конструктором та конструктором копіювання.

## Завдання 1 – Конструктор класу Dog

Програма:

* Додає до класу `Dog` конструктор, який приймає `std::string` та `int`.
* Дозволяє створювати об’єкти класу з певним ім’ям та віком, наприклад:

  ```cpp
  Dog fido {"Fido", 4};
  Dog spot {"Spot", 5};
  ```

## Завдання 2 – Конструктор копіювання

Програма:

* Додає до класу `Dog` конструктор копіювання.
* Конструктор копіювання створює новий об’єкт на основі існуючого та виводить:

  ```
  Copy Constructor
  ```
* Дозволяє створювати копії об’єктів:

  ```cpp
  Dog spot {"Spot", 5};
  Dog twin {spot}; // Викликає конструктор копіювання
  ```
